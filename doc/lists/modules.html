tty_mmod_tty.f08

Provide interactive menu using POSIX command stty. There is no use of the well known programming library Curses). Instead the command-line stty is used to set the terminal in raw mode in order to catch a unique key stroke without the need to press the key <Enter>. (This could be done with a call to libc to set canon / icanon.)
